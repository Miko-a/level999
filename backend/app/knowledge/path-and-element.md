# Path and Element

Source ID: path-and-element
Topic: path element basics
Game: Honkai: Star Rail
Version: internal-demo

In Honkai: Star Rail, each character has a Path and an Element.

A Path usually describes the character's combat role or design direction.

Common Path concepts:

- Destruction characters are often damage dealers with balanced offense and survivability.
- Hunt characters usually focus on single-target damage.
- Erudition characters usually focus on multi-target or area damage.
- Harmony characters usually support allies with buffs.
- Nihility characters often apply debuffs or damage-over-time effects.
- Preservation characters often protect allies with shields or defensive utility.
- Abundance characters usually provide healing or recovery.

An Element affects weakness breaking. If an enemy is weak to a character's Element, that character can reduce the enemy's toughness bar.

Important beginner advice:

- Path helps explain what a character usually does.
- Element helps decide whether a character can break enemy weakness.
- A good team considers both role coverage and enemy weakness.
