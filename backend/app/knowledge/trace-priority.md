# Trace Priority

Source ID: trace-priority
Category: progression
Topic: trace priority
Game: Honkai: Star Rail
Version: internal-demo
Tags: trace, skill, ultimate, talent, beginner, upgrade

Traces are character upgrades that improve abilities, stats, and combat performance.

For beginners, Traces are usually more reliable than farming perfect Relics. A character with upgraded important Traces often performs better than a character with weak Traces but random Relics.

General Trace priority depends on character role.

For main DPS characters:

1. Upgrade the ability that contributes most of their damage.
2. Upgrade Ultimate if it is a major damage source.
3. Upgrade Talent if the character relies on it.
4. Upgrade Skill if it is frequently used.
5. Basic Attack is usually lower priority unless the character relies on it.

For sustain characters:

1. Upgrade healing, shielding, or defensive utility first.
2. Upgrade Talent if it improves survival.
3. Upgrade Ultimate if it provides recovery or protection.
4. Damage-related upgrades are usually lower priority.

For support characters:

1. Upgrade buffing or debuffing abilities first.
2. Upgrade Ultimate if it provides team-wide value.
3. Upgrade Talent if it improves support uptime.
4. Damage upgrades are usually secondary.

Beginner warning:

Do not upgrade every Trace equally. Focus on the abilities that define the character's role.
