# Light Cone Basics

Source ID: light-cone-basics
Category: equipment
Topic: light cone basics
Game: Honkai: Star Rail
Version: internal-demo
Tags: light cone, weapon, beginner, upgrade, equipment

Light Cones are character equipment that provide base stats and passive effects.

A Light Cone should match the character's Path to activate its passive effect. For example, a Light Cone designed for a specific Path is usually most useful when equipped by a character with the same Path.

Beginner advice:

1. Use a Light Cone that matches the character's Path.
2. Level up the Light Cone used by the main DPS first.
3. After the main DPS, level up Light Cones for sustain and important supports.
4. Do not level too many Light Cones at the same time.
5. A properly leveled Light Cone is often more useful than chasing perfect Relics early.

Light Cone priority usually follows this order:

- Main DPS Light Cone
- Sustain Light Cone
- Core support Light Cone
- Other flexible characters

If a player feels weak, they should check whether their main DPS Light Cone is underleveled.
