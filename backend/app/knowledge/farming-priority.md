# Farming Priority

Source ID: farming-priority
Topic: farming priority
Game: Honkai: Star Rail
Version: internal-demo

For beginners, resource management is important. Players should avoid spreading resources across too many characters.

Recommended beginner farming priority:

1. Character level materials
2. Light Cone level materials
3. Important Trace materials
4. Ascension materials
5. Usable Relics
6. Better Relic substats later

Players should first build a core team instead of trying to build every character.

A good early goal is to build:

- One main DPS
- One sustain
- One or two supports

Relic farming should not become the main focus too early. A character with proper level, Light Cone, and Traces often performs better than a character with underleveled basics but slightly better Relics.

If the player feels weak, they should check:

- Is the main DPS leveled?
- Is the Light Cone leveled?
- Are important Traces upgraded?
- Is there a sustain character?
- Are Relic main stats usable?
